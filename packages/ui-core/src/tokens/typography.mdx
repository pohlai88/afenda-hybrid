# Typography

AFENDA typography pairs a **serif display face** with a **sans-serif body face**,
creating authority for headings while maintaining clarity for data-dense UIs.

---

## Font Families

| Token | Stack | Tailwind Class | Usage |
|-------|-------|----------------|-------|
| `--font-display` | Source Serif 4, Georgia, Cambria, serif | `font-display` | Headings, KPI values, page titles |
| `--font-sans` | Inter, system-ui, sans-serif | `font-sans` | Body text, labels, table data, UI elements |
| `--font-mono` | JetBrains Mono, Fira Code, Consolas | `font-mono` | Code, technical values |

> The **serif for headings / sans-serif for body** split signals authority while keeping interface
> text crisp and legible.

## Font Weights

| Weight | Tailwind | Usage |
|--------|----------|-------|
| 400 | `font-normal` | Body text, descriptions, table cells |
| 500 | `font-medium` | Labels, table headers, nav items |
| 600 | `font-semibold` | Card titles, dialog titles, badges, h3/h4 |
| 700 | `font-bold` | Page headings (serif), metric values, display |

---

## Line Heights

| Class | Value | Usage |
|-------|-------|-------|
| `leading-none` | 1 | Display metrics, large numbers |
| `leading-tight` | 1.25 | Headings, card titles |
| `leading-snug` | 1.375 | Compact body text |
| `leading-normal` | 1.5 | Default body text |
| `leading-relaxed` | 1.625 | Long-form descriptions |

## Usage Examples

```html
<!-- Page heading (serif, bold, tight tracking) -->
<h1 class="font-display text-[2rem] font-bold tracking-tight">
  Revenue grew 15% driven by enterprise segment
</h1>

<!-- Section title (serif) -->
<h2 class="font-display text-2xl font-semibold tracking-tight">
  Workforce Analytics
</h2>

<!-- Card title (sans-serif) -->
<h3 class="text-xl font-semibold">Open Positions</h3>

<!-- Body text -->
<p class="text-sm text-muted-foreground">
  Showing 1-20 of 143 employees
</p>

<!-- Overline label -->
<span class="text-[0.6875rem] font-semibold uppercase tracking-widest text-muted-foreground">
  Department
</span>

<!-- Caption / source attribution -->
<span class="text-xs text-muted-foreground">
  Source: HRIS export, March 2026
</span>
```

## Design Principles

1. **Serif for authority**: Use `font-display` on page titles and KPI values to
   signal importance. Never use serif for body text or labels.
2. **Round sensibly**: Display numbers at appropriate precision. Executives
   don't need two decimal places.
3. **Action titles**: Page headings should state conclusions, not topics.
   "Revenue grew 15%" not "Revenue Overview".
4. **Consistent hierarchy**: Follow the type scale strictly. Don't invent
   intermediate sizes.
